# Laravel Prototype 4

## Creation:

```bash
  composer create-project laravel/laravel Prototype_4

  cd Prototype_4

  php artisan key:generate

```

## Installation

```bash
  npm install admin-lte@^3.2.0

  composer require laravel/ui
  php artisan ui bootstrap --auth

  npm install
  npm run dev
```
